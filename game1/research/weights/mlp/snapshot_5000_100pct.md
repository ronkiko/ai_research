## Snapshot — Нейросеть (MLP) на Детектор лжи

- **Модель:** mlp (Нейросеть MLP)
- **Игра:** lie_detector (Детектор лжи)
- **Режим:** supervised
- **Шаги:** 5000
- **Точность:** 100%

### Условия
- Задача: XOR (два свидетеля, угадать, говорят ли одно и то же)
- Модель: MLP (2 входа → 4 скрытых ReLU → 1 выход)
- Режим: supervised (кросс-энтропия + Adam, LR=0.05)

### Архитектура

```
x0 ──╮
     ├──●──●──●──●──●──→ p
x1 ──╯   4 скрытых (ReLU)
```

### Веса первого слоя: Linear(2 → 4)

```
           нейрон 0  нейрон 1  нейрон 2  нейрон 3
weight[x0]   -0.257    -3.223    -0.697    +2.800
weight[x1]   -0.269    -3.226    -0.652    +2.801
bias         -0.225    +3.220    -0.256    -2.801
```

### Веса второго слоя: Linear(4 → 1)

```
           нейрон 0  нейрон 1  нейрон 2  нейрон 3
weight       +0.111    +5.925    +0.095    +6.671
bias         -8.088
```

### Как это работает (анализ по нейронам)

Скрытые нейроны 0 и 2 — «мёртвые» (их ReLU убит отрицательным смещением,
веса на выходе почти 0). Работают только нейроны 1 и 3.

**Нейрон 1:** `w = (−3.22, −3.22), b = +3.22`
- Сигнал: `ReLU(−3.22·x₀ − 3.22·x₁ + 3.22)`
- Горит только когда **оба входа = 0**: `−0 − 0 + 3.22 = 3.22`
- При (0,1): `−0 − 3.22 + 3.22 = 0` → ReLU гасит
- При (1,0): `−3.22 − 0 + 3.22 = 0` → ReLU гасит
- При (1,1): `−3.22 − 3.22 + 3.22 = −3.22` → ReLU гасит
- **Смысл: «оба сказали нет»**

**Нейрон 3:** `w = (+2.80, +2.80), b = −2.80`
- Сигнал: `ReLU(2.80·x₀ + 2.80·x₁ - 2.80)`
- Горит только когда **оба входа = 1**: `2.80 + 2.80 − 2.80 = 2.80`
- При (0,0): `0 + 0 − 2.80 = −2.80` → ReLU гасит
- При (0,1): `0 + 2.80 − 2.80 = 0` → ReLU гасит
- При (1,0): `2.80 + 0 − 2.80 = 0` → ReLU гасит
- **Смысл: «оба сказали да»**

**Выход:** сумма сигналов × веса + смещение = `5.93·нейрон1 + 6.67·нейрон3 − 8.09`

| Входы  | Нейрон 1 | Нейрон 3 | logit     | p      | Решение |
|--------|----------|----------|-----------|--------|---------|
| (0,0)  | 3.22     | 0        | +10.99    | 1.000  | правда ✓|
| (0,1)  | 0        | 0        | −8.09     | 0.000  | ложь ✓  |
| (1,0)  | 0        | 0        | −8.09     | 0.000  | ложь ✓  |
| (1,1)  | 0        | 2.80     | +10.59    | 1.000  | правда ✓|

### Суть

MLP «изобрёл» два детектора: **«оба сказали нет»** и **«оба сказали да»**.
Когда не сработал ни один — значит показания расходятся, и выходной нейрон
голосует «ложь» за счёт отрицательного смещения. Это ровно то, как человек
описал бы правило: «правда, когда показания совпадают; ложь — когда нет».

Два других нейрона (0 и 2) модель просто выключила — они не нужны. То есть
4 скрытых нейрона — с запасом, хватило бы и 2 (если б заранее знать, какие).

### Воспроизведение

`torch.manual_seed(4)` даёт именно эти веса.
