## Нейросеть (MLP)

**2 входа → 4 скрытых (ReLU) → 1 выход** — решает XOR.

### Схема

```
x0 ──╮
     ├──●──●──●──●──●──→ p
x1 ──╯    скрытый слой
```

### Параметры

- 8 весов на первый слой (2 входа × 4 нейрона)
- 4 bias'а на скрытые нейроны
- 4 веса на выходной слой (4 нейрона × 1 выход)
- 1 bias на выход

Итого 17.

Формула:

```
h = ReLU(W₁·x + b₁)     # 4 нейрона
p = σ(W₂·h + b₂)         # 1 выход
```

Нейросеть с одним скрытым слоем. В отличие от линейных моделей, может
выучить XOR — задачу, где ответ зависит не от каждого входа по отдельности,
а от их комбинации. Градиенты считает PyTorch (autograd), оптимизатор Adam.
