## Контекстный нейрон

**4 параметра (3 входа + смещение)** — для игр с контекстом.

### Схема
            
    x0·w0 ──┐
    x1·w1 ─═╬═─ σ ──→ p
    x2·w2 ──┤
    b     ──╯

### Формула

    logit = w0·x0 + w1·x1 + w2·x2 + b

Модель видит три последних исхода и взвешивает каждый своим весом.
Если в мире есть скрытый ритм длиннее одного шага — контекстный нейрон
может его разгадать. Подходит для Pattern.
