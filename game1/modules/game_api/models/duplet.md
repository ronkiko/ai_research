## Нейрон-дуплет

**3 параметра (2 входа + смещение)** — для игр с двумя входами.

### Схема

    x0·w0 ──╮
    x1·w1 ──╫── σ ──→ p
    b     ──╯

### Формула

    logit = w0·x0 + w1·x1 + b

Два входа — ровно столько нужно, чтобы разгадать двойное чередование [0,0,1,1].
Первый вес отвечает за то, был ли прошлый исход единицей, второй — за позапрошлый.
По сравнению с Logistic добавляет один вход, по сравнению с Context — на один
меньше (достаточно для паттернов длиной до 4 шагов).
