## Один нейрон с входом

**2 параметра (w + b), один вход** — для игр с состоянием.

### Схема

    x·w ──╮
    b   ──╫── σ ──→ p
          

### Формула

    logit = w·x + b

Модель видит последний исход. Вес w решает, насколько часто мир повторяется:
положительный — мир любит оставаться на месте, отрицательный — любит прыгать.
Подходит для Kormushka, Dealer.
